# Менеджер книг (Book Manager)

Приложение для учета личной библиотеки. Позволяет добавлять книги, сохранять их в JSON и искать по названию или автору.

## Требования

- Python 3.13+
- Стандартные библиотеки: tkinter, json, unittest

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Always-in-prime/book-manager.git
   cd book-manager
